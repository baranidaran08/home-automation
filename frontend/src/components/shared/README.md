# Shared Components

App-wide reusable components that are **not** primitive shadcn/ui elements —
things composed from `ui/` primitives and used across multiple features, e.g.
`PageHeader`, `DataTable`, `EmptyState`, `ConfirmDialog`, `LoadingSpinner`.

- `components/ui/` — shadcn/ui primitives (button, card, input, …).
- `components/shared/` — composed, app-specific reusable components.
